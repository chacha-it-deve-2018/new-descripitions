## 1. 📚 C++ DBMS (Stack Data Structure)
A console-based Database Management System built using the **Stack (LIFO)** data structure. This project demonstrates core data manipulation logic.

### ✨ Key Features
* ➕ **Add Record (Push):** Efficiently push new data onto the stack.
* ➖ **Delete Record (Pop):** Manage records by removing the most recent entry.
* 🔍 **Search & Update:** Locate and modify records using specific unique IDs.
* 📋 **Display:** View the entire stack of records in a clean format.

### 🛠️ Technical Stack
* **Language:** C++
* **Core Logic:** LIFO (Last-In-First-Out) Principle
