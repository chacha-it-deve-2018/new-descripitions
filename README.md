## 1. ­ЪЊџ C++ DBMS using Stack Data Structure
рІГрѕЁ рЇЋрѕ«рїЇрѕФрѕЮ рІе **Stack (LIFO)** рІ│рЅ│ рѕхрЅхрѕФріГрЅИрѕГріЋ рЅарѕўрїарЅђрѕЮ рѕўрѕерїЃріЋ ріЦріЋрІ┤рЅх рѕЏрѕхрЅ░рІ│рІ░рѕГ ріЦріЋрІ░рѕџрЅ╗рѕЇ рІФрѕ│рІФрѕЇрЇб

* **Add Record (Push):** ріарІ▓рѕх рѕўрѕерїЃ рІѕрІ░ рѕхрЅ│ріГ рѕўрїерѕўрѕГрЇб
* **Delete Record (Pop):** рІерѕўрїерѕерѕ╗рІЇріЋ рІерїѕрЅБ рѕўрѕерїЃ рѕЏрїЦрЇІрЅхрЇб
* **Search & Update:** рѕўрѕѕрІФ рЅЂрїЦрѕГріЋ (ID) рЅарѕўрїарЅђрѕЮ рѕўрѕерїЃ рѕўрЇѕрѕѕрїЇ ріЦріЊ рѕЏрѕхрЅ░ріФріерѕЇрЇб
* **Display:** рѕЂрѕЅріЋрѕЮ рѕўрѕерїЃрІјрЅй рЅарІЮрѕГрІЮрѕГ рѕЏрІерЅхрЇб



### ­ЪЏа№ИЈ рЅ┤ріГріќрѕјрїѓ (Tech Stack)
* **Language:** C++
* **Logic:** Last-In-First-Out (LIFO) principle.
